快速入门命令

如果你有闲工夫，请看指南

/cd（或者右键你快捷栏里那个红色的书）打开菜单（菜单就是把命令用GUI呈现，对不熟悉命令的新人更友好）

cmi插件

/cmi dback 返回你上次死亡的位置

/cmi back 返回你上次传送的地方（可以跨维度）

/cmi sethome 名称 设置你的家 （名称可以用数字或字母）（在人物所在处设立传送点，后续可以通过指令传送回来，每个人只能设立2个家

/cmi homes 管理你的家（你会看到如下图所示的界面，方便快捷）

/cmi removehome 名称 删除你的家（名称可以用数字或字母）（如果你已经有两个家却还想再建立一个家，必须先删除一个）（赞助服务器可以提升这个限制）

/tpa 玩家A 你自己 把你传送到玩家A的位置 （需要对方同意，且过程中不能移动或者被攻击）（下图是玩家A视角，需要打开命令栏再用鼠标点击即可同意或拒绝）

lands领土插件

按F3+G显示区块边界，这有助于划你的领地。
使用/lands help可以帮助你掌握这个插件的用法。

在菜单（/cd）中点击长得像萝卜一样的玩意可以方便的使用领土插件。

下面没有列出的操作基本都可以在菜单中进行操作。

/lands createcamp 名称 创建营地（营地是临时领土，过一段时间会被删除）

/lands create 名称 创建永久领土 一定要给你的家创建领土！！

/lands map 用GUI认领区块,这是一个非常方便的功能，可以防止使用”/lands claim”认领的区块不相连.

/lands viem 用粒子效果显示自己或他人领土的范围

/lands unclaim 取消认领所站的区块，会退还一定的游戏币

如何删除领土：

  1. 首先，/lands edit 名称 选择要编辑的领土
  2. 然后，/lands delete 删除刚刚选择的领土

注意事项：

  1. 领土和国家都是有维护费的,维护费由服务器定期收取,如果你是国家级别的领土那你将承受全部加入你的领土维护费
  2. 你的领土如果信任更多的玩家将会提高领土的规模(领土区域区块的上限,或许还有减少维护费),每个信任的玩家也会提高领土可认领的区块上限.
  3. 如果想建立国家规模的领土,需要领土升到最高级,然后邀请其他领土加入可以提高国家级别.

服务器的历史与未来

历史

这是一个老破小服务器,2019年9月14日开始搭建大概一周后开始宣传,服务器具有悠久历史与故事,服务器从Mincecraft 1.14版本开始到最新版本历经数个周目都没有跑路,本服完全由服主一人管理与增加内容(玩法)。

为什么转型成商业服

因为服务器在1.16.5版本时发现各种高质量的插件和维护的开销都不小,杜绝在玩家的服务器上使用盗版和泄露的插件,以防插件导致不稳定甚至后门,同时也方便寻找插件作者进行售后维护.**服务器需要电费网费等等的基础开销,不可能一直公益下去,**不打着公益的旗帜做商业,除了最开始1.14版本宣传服务器是公益类型后全部改成了商业类型.虽然服务器是商业类型,但氪金与不氪金的差距不会很大.保证全体玩家的游玩体验,也不会因为玩家赞助了进行偏护行为(因此损失不少富哥)

对于未来

目前服务器在往轻RPG和更多内容玩法上发展,俗称的插件服并非纯粹的生电服,当然也不会一刀切完全不准生电行为,会进行一些限制,属于半生电服.往轻RPG和更多内容上发展可以大大的提高服务器的寿命,我的世界这款游戏自身的内容实际上并不是非常多,核心在创造性上建各种设备房屋等等,但这是会慢慢觉得枯燥,只有添加游戏的内容才能让游戏的寿命更加长久.

关于周目

  1. 一个周目大概有3到4个月，每个周目的长度都不一样
  2. 新周目会重置任何东西，你的资产、领地、游戏币、地图等等…
  3. 一个周目的末期可能会有模组服可以玩
  4. 一般来说新周目的第一天就会有人走完原版主线把龙打死了，龙蛋可能也会被拿走

注意事项

  • 服务器不会对任何挂机状态的玩家任何情况的死亡和损失负责，挂机请在安全的地方
  • **圈好自己的地盘！**如果不圈地的情况下放在领土外的东西被拿走，服主和管理均不负责.
  • 本服是死亡掉落,无法保证玩家在死亡后的装备能否回到它的原主人身上.
  • 请注意玩家交易需谨慎，如果明确对方是个骗子，可以在服务器或者群里面反应，会有人把他头拧下来的

插件钓鱼的四种方式

其实是四种小游戏

  • 此类型中 点按shift使鱼保持在绿条中 等待鱼钩的边框攒满就行了
  • 此类型中 游标会在条上左右移动，如果游标指到了条上大字所要求的颜色，右键即可钓鱼（示意图为蓝色）
  • 此类型和上一个类型差不多，游标指到绿色（只有绿色可以）时右键就行
  • 此类型很难 点按shift让鱼通过终点线的同时不要让鱼竿框红满了 不然鱼就跑了

矛盾处理

物品纠纷

  • 若玩家间发生物品纠纷，例如未圈地保护的物品被他人拿走，服主会因无圈地记录而拒绝查询及退还。
  • 若失物者明知拿走物品的人并实施无限恶意报复，服主将介入处理——因未圈地导致物品丢失的主要责任在自身，但多次甚至无限恶意报复则属过当行为。

处罚制度

所有的连续触犯,将不警告,直接处罚

  • 任何有目的性的辱骂谩骂先警告2次,第三次直接游戏内禁言15分钟.
  • 任何恶意故意挑衅其他玩家进行人身攻击的警告2次,第三次直接关30分钟小黑屋(被关者下线不计时).
  • 任何恶意故意引诱击杀迫害萌新的新老玩家警告1次,第二次直接关3小时小黑屋(被关者下线不计时).

规则

禁止事项

基本

  1. 服务器禁止任何玩家通过作弊所带来的优势行为.(允许2个小号.)
  2. 服务器禁止任何玩家恶意攻击甚至击杀萌新.(萌新没有装备,在中后期装备差距过大,一但出现这种情况,会严惩)
  3. 服务器禁止玩家进行任何恶意的卡服或者利用BUG的行为.(无法保证任何的版本完全没有漏洞,插件也是,难免会有疏忽大意，这也是服务器为什么时不时要更新核心和插件,不更新的服务器一定不是好服务器)
  4. 服务器禁止玩家进行刷屏与辱骂行为。服主有权禁言恶劣者甚至踢出!!!(玩家之间肯定会有矛盾,服主只能限制一些容易造成大量矛盾的行为.)
  5. 如果有人引起公愤，服主不会介入。

禁用的命令

/seed /gamemode /

因服务器性能而做出的限制

一些行为会导致服务器卡顿，使得其他玩家的游戏体验变差，所以要避免。

  • 服务器会限制使服务器变得卡顿的设备（比如刷怪、生电）
  • 不要有太多的实体（生物、掉落物、盔甲架、经验球、展示框、船、矿车等）挤压在一起

关于赞助

赞助可以获得一些不影响游戏平衡的权限。
**服务器赞助主要分为3种，直接购买点卷、购买减负包、参与众筹。**前两种可以在游戏内直接输入/菜单 界面右下角点击赞助菜单进入赞助界面。

  • 点卷：一般的权限购买代币，可以用来兑换（时长一般为一个月：
    • 游戏币
    • 功能性或装饰性权限 （聊天彩色字体，更多附魔查询，称号装饰，上架物品增加等）
  • 月卡（减负包）：减负包是按月支付
    • 一次性获得200点卷
    • 每日可以领取800游戏币和一个不死图腾
    • 减负包持续期间圈地价格减少0.1 ,领地上限+1
    • mcmmo经验获取增加10%
  • 众筹：当服务器需要添加插件时会开放众筹，由多名玩家一起赞助购买插件所需要的金额，参与众筹后会进入众筹组。
    • 增加领地及设家上限
    • 并给予一些插件的使用权限，但只会多部分娱乐及功能性的权限，不会破坏玩家间的游戏平衡，如cmi的装饰性权限(/ride，/hat，盔甲架编辑)等
